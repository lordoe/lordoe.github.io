Reinforcement Learning

#reinforcement

Ein Agent lernt durch Interaktion mit einer Umgebung. Er erhält nach jeder Aktion eine Belohnung (Reward) und optimiert seine Policy, um den kumulierten zukünftigen Reward zu maximieren.

Kernkonzepte: Policy π, Wertfunktion V(s), Aktionswertfunktion Q(s,a), Bellman-Gleichungen, Exploration vs. Exploitation, Temporal Difference Learning.

📄 Reinforcement Learning

⤢ k-Armed Bandit Greedy, ε-greedy, UCB und optimistische Initialwerte — Exploration-Exploitation live simuliert. // interaktiv
⤢ Q-Learning & SARSA TD-Learning in einer Grid World — Q-Tabelle wächst live, Policy emergiert aus Episoden. // interaktiv
⤢ Actor-Critic & REINFORCE TD-Loss, K-step Bellman, Log-Derivative Trick, Advantage-Funktion — animierter AC-Loop. // interaktiv
