Principal Component Analysis

Schritt-für-Schritt Visualisierung · 2D → 1D Dimensionsreduktion

// 1D PCA — grundidee

Zentriere die Daten: \(\tilde{\mathbf{x}}_n = \mathbf{x}_n - \bar{\mathbf{x}}\). Finde den Einheitsvektor w ∈ ℝd mit maximaler Varianz der Projektion:

\[ \mathbf{w}^* \;=\; \arg\max_{\|\mathbf{w}\|=1} \;\frac{1}{N}\sum_{n=1}^{N} \bigl(\mathbf{w}^\top \tilde{\mathbf{x}}_n\bigr)^2 \]

w₁ ⊥ w₂: jede weitere Komponente steht senkrecht auf allen vorherigen.

// multi-dimensionale PCA

Finde K orthonormale Richtungen w₁, …, w_K ∈ ℝd mit maximaler Gesamtvarianz:

\[ \max_{\mathbf{w}_1,\dots,\mathbf{w}_K} \sum_{i=1}^{K} \mathrm{Var}(\mathbf{w}_i^\top \mathbf{x}) \quad \text{s.t.} \quad \|\mathbf{w}_i\|=1 \;\forall i,\quad \mathbf{w}_i \perp \mathbf{w}_j \;\forall i\neq j \]

Die K-dimensionale PCA-Repräsentation ist dann:

\[ \mathbf{z}_n \;=\; \mathbf{W}^\top \tilde{\mathbf{x}}_n \;\in\; \mathbb{R}^K, \qquad \mathbf{W} = [\mathbf{w}_1, \dots, \mathbf{w}_K] \in \mathbb{R}^{d \times K} \]

// PCA in Kovarianzform

Empirischer Mittelwert und Kovarianzmatrix:

\[ \bar{\mathbf{x}} \;=\; \frac{1}{N}\sum_{n=1}^N \mathbf{x}_n, \qquad \mathbf{C} \;=\; \frac{1}{N}\sum_{n=1}^N (\mathbf{x}_n - \bar{\mathbf{x}})(\mathbf{x}_n - \bar{\mathbf{x}})^\top \;\in\; \mathbb{R}^{d\times d} \]

Da \(\sum_i \mathrm{Var}(\mathbf{w}_i^\top \mathbf{x}) = \mathrm{tr}(\mathbf{W}^\top \mathbf{C}\mathbf{W})\), ist PCA äquivalent zu:

\[ \max_{\mathbf{W} \in \mathbb{R}^{d\times K}} \mathrm{tr}(\mathbf{W}^\top \mathbf{C}\mathbf{W}) \quad \text{s.t.} \quad \mathbf{W}^\top \mathbf{W} = \mathbf{I}_K \quad \text{(orthogonale Spalten)} \]
▸  Die Top-K Eigenvektoren der Kovarianzmatrix C = Cov(x) sind die geschlossene Lösung.
▸  Äquivalent kann die SVD der zentrierten Datenmatrix sie bestimmen.

// erklärte Varianz

Die Eigenwerte λ₁ ≥ … ≥ λ_d von C geben die durch jede Hauptkomponente erklärte Varianz an:

\[ \text{(Erklärte Varianz der Top-}K\text{)} \;=\; \frac{\sum_{k=1}^{K} \lambda_k}{\sum_{k=1}^{d} \lambda_k} \]

Der Scree-Plot zeigt diese kumulierte Varianz — hilft die optimale Anzahl K zu wählen.

// algorithmus (pseudo-code)

# x hat Form N × d
x = x - x.mean(axis=0)      # Daten zentrieren
C = (x.T @ x) / N           # empirische Kovarianz, Form (d, d)
lam, W = eigh(C)            # Eigenzerlegung (aufsteigend nach lam)
W = W[:, ::-1][:, :K]       # Top-K Eigenvektoren, Form (d, K)
z = x @ W                   # K-dim Repräsentation, Form (N, K)
