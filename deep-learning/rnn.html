RNN — Recurrent Neural Networks

// Deep Learning · Foliensatz 08
01 — Recurrent Neural Network

Eine Recurrent Neural Network (RNN) verarbeitet Sequenzen \(\mathbf{x}_1, \mathbf{x}_2, \dots, \mathbf{x}_T\), indem es einen versteckten Zustand \(\mathbf{y}_t\) über die Zeit fortschreibt: \(y_t = \text{RNN}(x_t,\, y_{t-1})\).

ReLU-RNN
\[ \mathbf{y}_t = \text{ReLU}\!\bigl(\mathbf{W}_{\text{rec}}\,\mathbf{y}_{t-1} + \mathbf{W}_{\text{in}}\,\mathbf{x}_t + \mathbf{b}\bigr) \]
\(\mathbf{W}_{\text{rec}}\) — Rekurrente Gewichte  \((D' \times D')\)
\(\mathbf{W}_{\text{in}}\) — Input-Gewichte  \((D \times D')\)
\(\mathbf{b}\) — Bias-Vektor  \((D',)\)
Einfaches Beispiel — Exponentieller gleitender Mittelwert
\[ y_t = \alpha\, y_{t-1} + x_t \]

Für \(\alpha \in (0,1)\) akkumuliert das Netzwerk Informationen vergangener Zeitschritte mit exponentiell abfallendem Gewicht.

Interaktiv: Unrolled RNN
Drücke Step Forward, um die Verarbeitung Schritt für Schritt zu verfolgen.
Kausales Rezeptionsfeld
\(y_t\) hängt von \(x_1, x_2, \dots, x_t\) ab — alle vergangenen Eingaben. Vergleich Conv1D: endliches Rezeptionsfeld \(1 + L(K-1)\).
02 — Vanishing / Exploding Gradients

Betrachte ein lineares RNN: \(\mathbf{y}_t = \mathbf{W}\,\mathbf{y}_{t-1}\), mit \(\mathbf{y}_0 = \mathbf{x}_0\). Sei \(\lambda\) ein Eigenwert von \(\mathbf{W}\) mit Eigenvektor \(\mathbf{v}\).

Exploding (Vorwärts)
Störung \(\varepsilon\mathbf{v}\) an \(\mathbf{x}_0\):
\(\|\mathbf{y}_T - \mathbf{y}'_T\| = \varepsilon|\lambda|^T\)

Wenn \(|\lambda| > 1\): Störungen wachsen exponentiell.
Vanishing (Rückwärts)
Kettenregel rückwärts:
\(\frac{\partial L}{\partial \mathbf{y}_{t-1}} = \mathbf{W}^\top \frac{\partial L}{\partial \mathbf{y}_t}\)

Wenn \(|\lambda| < 1\): Gradient nach \(T\) Schritten \(\approx |\lambda|^T \to 0\).
Interaktiv: Gradientenmagnitude
Eigenwertbetrag \(|\lambda|\) 0.90
Sequenzlänge \(T\) 10
03 — Long Short-Term Memory (LSTM)

Das LSTM löst das Vanishing-Gradient-Problem durch explizite Memory-Zellen \(c_t^k\), die über Gatter kontrolliert werden.

Memory-Zelle
\[ c_t^k = f_t^k\, c_{t-1}^k + i_t^k\, c_t^{\text{new},k} \]

\(f_t^k \in [0,1]\): Forget Gate  ·  \(i_t^k \in [0,1]\): Input Gate

Sichtbare Einheiten
\[ v_t^k = o_t^k\, \sigma(c_t^k) \]

\(o_t^k \in [0,1]\): Output Gate

Gatter-Berechnungen
\[ \mathbf{i}_t, \mathbf{f}_t, \mathbf{o}_t = \sigma\!\bigl(\text{Linear}([\mathbf{x}_t, \mathbf{v}_{t-1}])\bigr) \] \[ \mathbf{c}_t^{\text{new}} = \tanh\!\bigl(\text{Linear}([\mathbf{x}_t, \mathbf{v}_{t-1}])\bigr) \]
04 — RNN vs. Conv1D Vergleich

Vergleich der wichtigsten Eigenschaften von RNNs und konvolutionalen 1D-Netzwerken:

Eigenschaft RNN Conv1D
Rezeptionsfeld unendlich (alle Vergangenheit) endlich \(1 + L(K-1)\)
Parallelisierbar nein (sequentiell) ja
Nicht-linear ja nein (linear)

\(L\): Anzahl Layer · \(K\): Kernelgröße
